# Trabajo Práctico en MySql - Micaela Asselborn

## Desarrollo Web - Back End - UTN

## Objetivo

Implementar un sistema de gestión para la veterinaria "Patitas Felices" demostrando:

- Diseño de esquema relacional con integridad referencial
- Operaciones CRUD completas
- Consultas complejas con JOINs
- Gestión de transacciones y eliminaciones en cascada

## Descripción

En este trabajo creo una
base de datos llamada "veterinaria_patitas_felices" dónde almaceno los datos de dueños, mascotas, veterinarios y consultas clinicas mediante tablas que se relacionan con claves foranéas.

## Operaciones CRUD implementadas

### **C - CREATE** (Crear)

- Base de datos y estructura de tablas
- Relaciones con claves foráneas
- Índices para optimización

### **R - READ** (Leer)

- Consultas simples y avanzadas
- JOINs entre múltiples tablas
- Ordenamientos y filtros

### **U - UPDATE** (Actualizar)

- Modificación de registros existentes
- Actualización de datos relacionados
- Transacciones para consistencia

### **D - DELETE** (Eliminar)

- Eliminación con integridad referencial
- ON DELETE CASCADE para eliminación automática
- Verificación de impactos
